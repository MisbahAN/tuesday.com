### Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/MisbahAN/tuesday.com.git
   cd tuesday.exe
   ```

2. **Install dependencies**
   ```bash
   npm install
   npm install dotenv
   ```

3. **Start development server**
   ```bash
   node app.js
   ```
