### Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/MisbahAN/tuesday.com.git
   cd tuesday.exe
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm start
   ```

4. **Run tests**
   ```bash
   npm test
   ```

5. **Create production build**
   ```bash
   npm run build
   ```
